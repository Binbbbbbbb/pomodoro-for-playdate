# Pomodoro for Playdate

一款为 Playdate 打造的像素风格番茄钟，支持专注计时、休息提醒与自动循环。

## 功能特色

**时钟首页** — 默认显示当前时间与日期。

<p align="center">
  <img src="assets/images/clock.png" alt="时钟首页" width="400">
</p>

**工作计时** — 默认 25 分钟专注计时，按 `A` 键开始。

<p align="center">
  <img src="assets/images/worktime.png" alt="工作计时" width="400">
</p>

**休息计时** — 默认 5 分钟休息计时，工作阶段结束后自动进入。

<p align="center">
  <img src="assets/images/breaktime.png" alt="休息计时" width="400">
</p>

**循环番茄钟** — 按「工作 → 休息」节奏自动循环，默认 3 轮。

**自定义设置** — 可分别调整工作时长、休息时长与循环次数。

<p align="center">
  <img src="assets/images/settings.png" alt="自定义设置" width="400">
</p>

**主题切换** — 支持深色 / 浅色两种显示主题。

<p align="center">
  <img src="assets/images/theme-dark.png" alt="深色主题" width="400">
</p>

<p align="center">
  <img src="assets/images/theme-light.png" alt="浅色主题" width="400">
</p>

**完成提醒** — 每个阶段计时结束后弹出提示并播放提示音。

<p align="center">
  <img src="assets/images/done.png" alt="完成提醒" width="400">
</p>

**曲柄交互** — 设置页转动曲柄可调整数值，计时页转动曲柄可重置当前计时。

## 操作说明

| 操作 | 说明 |
|------|------|
| 方向键 | 在时钟、工作计时、休息计时、设置、主题页面之间切换 |
| `A` 键 | 开始 / 暂停计时；主题页切换主题 |
| `A` 键（长按） | 设置页恢复默认值 |
| 曲柄 | 设置页调整数值；计时页重置当前计时 |
